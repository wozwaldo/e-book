# e-book
A frontend book landing page with a form and animated elements.


## Features
- Book landing page design
- Subscription form with name and email input
- Animated bubbles that float from the bottom to the top
- Responsive design


## Tech Stack
- HTML5
- CSS3
- JavaScript
